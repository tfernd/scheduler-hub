# Scheduler Hub

Scheduler Hub is a Github repository that contains various schedulers for text-to-image models, with a focus on stable-diffusion. The goal of this project is to make it easier for users to compare and experiment with different schedulers, as well as to combine them for enhanced performance.

# Installation

To install Scheduler Hub, simply clone this repository to your local machine:

```bash
pip install git+https://github.com/tfernd/scheduler-hub.git
```
